## Backend - Projeto de Lista de Empresas

### Tecnologias utilizadas:

- Java 11
- Spring Boot
- Spring Web Tools
- Spring Data Jpa
- MySQL
- H2 Database

- [x] ● Configuração do projeto em https://start.spring.io/
